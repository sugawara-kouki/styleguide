ソースファイルの設定

ファイルのエンコード：UTF-8

ソースファイルはUTF-8でエンコードされていること

空白文字

空白文字については全角のものを使用しない事とする。ただし、リテラル文字列内のみ使用することを許可する

コーディング規約

変数と関数

変数と関数名にはcamelCaseを使用します。これは従来のJavaScriptと同様です

悪い例
var FooVar;
function BarFunc() {}
良い例
var fooVar;
function barFunc() {}

クラス

クラス名にはPascalCaseを使用します。これは従来のJavaScriptで一般的なルールです

悪い例
class foo { }
良い例
class Foo { }

クラスメソッド、メンバ変数についてはcamelCaseを使用します。

悪い例
class Foo {
  Bar: number;
  Baz() { }
}
良い例
class Foo {
  bar: number;
  baz() { }
}

インターフェース

インターフェース名はPascalCase、メンバはcamelCaseを使用します。

プレフィックスにIをつけないでください。

悪い例
interface IFoo {
}
良い例
interface Foo {
}

タイプ

名前はPascalCase、メンバはcamelCaseを使用します。

名前空間

名前はPascalCaseを使用します。

悪い例
namespace foo { }
良い例
namespace Foo { }

Enum

enum名はPascalCaseを使用します。

悪い例
enum color {
}
良い例
enum Color {
}

enumのメンバはPascalCaseを使用します。

悪い例
enum Color {
  red
}
良い例
enum Color {
  Red
}
